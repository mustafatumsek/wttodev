Bu web sitesini "Web Tabanlı Teknolojiler" dersim için proje olarak hazırladım. Bir fotoğrafçının portfolyo sitesi olarak oluşturduğum sitede fotoğraflar, hakkımda ve iletişim olmak üzere 3 ana kısım bulunuyor. Fotoğraflar kısmında portfolyo içindeki fotoğraflar görüntülenebilir, tam ekranda görüntülenebilir ve açıklama yazıları okunabilir. Hakkımda kısmında biyografiye, iletişim bilgilerine ve teknik becerilere erişilebilir. İletişim kısmında ise mesaj bölümü yer alıyor. Bunlara ek olarak footerda bilgi, blog ve etiketler kısmı bulunmakta.
